# Mathematical Modeling Log

A personal learning repository for studying and practicing mathematical models for business applications, with a focus on user behavior analysis and A/B testing effectiveness in e-commerce.

## Purpose

This repository serves as a learning log and practice space for exploring statistical models and analytical techniques relevant to data-driven business decisions. The primary focus areas include:

- **User Behavior Analysis**: Understanding and modeling customer actions, preferences, and patterns
- **A/B Testing**: Statistical methods for evaluating the effectiveness of changes and experiments
- **E-commerce Analytics**: Applications of mathematical models to online retail scenarios

## Repository Contents

This repository contains:

1. **Markdown Documentation**: Summaries of statistical models, analysis techniques, and theoretical concepts
2. **Python Analysis Code**: Sample implementations and practical examples using modern data science libraries

## Tech Stack

The code examples in this repository primarily use:

- **Python 3**: Primary programming language
- **Pandas**: Data manipulation and analysis
- **Scikit-learn**: Machine learning and statistical modeling
- **Statsmodels**: Statistical tests and econometric models
- **Jupyter Notebooks**: Interactive analysis and documentation (when applicable)

## Getting Started

### Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/shuhey-hashimoto/Mathematical-Modeling-Log.git
   cd Mathematical-Modeling-Log
   ```

2. Create a virtual environment (recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install required packages:
   ```bash
   pip install pandas scikit-learn statsmodels jupyter matplotlib seaborn
   ```
